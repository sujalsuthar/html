Computer Parts

A typical computer consists of several important components:

  • Central Processing Unit (CPU): The brain of the computer.
  • Memory (RAM): Stores data temporarily for the CPU to process.
  • Hard Drive (HDD/SSD): Stores permanent data and the operating system.
  • Motherboard: The main circuit board connecting all components.
  • Graphics Processing Unit (GPU): Handles graphics and video rendering.
  • Power Supply Unit (PSU): Supplies power to all components.
  • Peripherals: Input and output devices like keyboard, mouse, and monitor.

These are some of the essential parts of a computer.
